DRAWING IN JavaScript

I created this drawing of a fox using JavaScript, exploring the capabilities of the language to generate dynamic graphics. For this, I used the canvas element and focused the development on geometric shapes. This project allowed me to deepen in visual programming, the use of coordinates and the manipulation of graphic elements through code. It was also an excellent exercise to combine artistic creativity with technical skills in web development.
