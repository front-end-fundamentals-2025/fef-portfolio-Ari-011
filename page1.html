CAMPUS RUSH

I developed a retro video game in JavaScript, in which the player controls a pixelated character that must collect books while interacting with NPCs that provide clues to find his classroom, the character can get more speed if he collects coffees and if he collects more than 5 coffees he becomes slower. I implemented pixel art graphics, the game contains NPC interaction and collection mechanics. The project included the use of Adobe Illustrator for the characters and the Tile platform to make the game map. This project allowed me to improve my programming skills.
